Abaixo estão algumas ideias de **instrumentalização para hinos da CCB em estilo orquestra**, usando os SF2 que você comentou: **CrisisGeneralMidi301.sf2**, **GeneralUser-GS.sf2** e **SGM-V2.01.sf2**.

A ideia é manter o som **sacro, limpo, reverente e congregacional**, sem parecer trilha de cinema exagerada.

---

## 1. Orquestra CCB clássica e equilibrada

| Voz       | Instrumento                |
| --------- | -------------------------- |
| Soprano   | Flauta ou Violino          |
| Contralto | Clarinete ou Trompa        |
| Tenor     | Viola ou Violoncelo        |
| Baixo     | Violoncelo grave ou Fagote |

**Fica bom para:** praticamente qualquer hino.
**Som esperado:** suave, bonito e bem tradicional.

**SF2 indicado:**
Use **SGM-V2.01** para cordas e madeiras. Se quiser som mais encorpado, teste o **CrisisGeneralMidi301** nas cordas.

---

## 2. Orquestra com órgão de fundo

| Voz       | Instrumento principal | Apoio                 |
| --------- | --------------------- | --------------------- |
| Soprano   | Flauta/Violino        | Church Organ suave    |
| Contralto | Clarinete             | Church Organ suave    |
| Tenor     | Viola/Cello           | Church Organ suave    |
| Baixo     | Cello/Fagote          | Pipe Organ grave leve |

**Fica bom para:** hinos de culto, hinos solenes e gravações mais próximas do ambiente da CCB.
**Som esperado:** parece uma orquestra acompanhada por órgão.

**Dica importante:** o órgão não deve dominar. Ele deve ficar como “cola harmônica” atrás da orquestra.

---

## 3. Metais suaves, estilo banda sacra

| Voz       | Instrumento                  |
| --------- | ---------------------------- |
| Soprano   | Trompete suave ou Flugelhorn |
| Contralto | Trompa                       |
| Tenor     | Eufônio ou Trombone suave    |
| Baixo     | Tuba leve                    |

**Fica bom para:** hinos fortes, majestosos e de exaltação.
**Som esperado:** cheio, nobre e congregacional.

**Cuidado:** não deixe o trompete muito alto, senão fica agressivo.
O melhor é usar **trompete com volume controlado** e **trompa/eufônio preenchendo o meio**.

---

## 4. Cordas completas, bem emocional

| Voz       | Instrumento                          |
| --------- | ------------------------------------ |
| Soprano   | Violino 1                            |
| Contralto | Violino 2 ou Viola                   |
| Tenor     | Viola ou Violoncelo                  |
| Baixo     | Violoncelo grave ou Contrabaixo leve |

**Fica bom para:** hinos lentos, de súplica, oração e comunhão.
**Som esperado:** emocionante, macio e profundo.

**SF2 indicado:**
O **CrisisGeneralMidi301** pode ficar bonito nas cordas, mas precisa controlar volume e reverb. O **SGM-V2.01** costuma ser mais equilibrado.

---

## 5. Madeiras delicadas

| Voz       | Instrumento               |
| --------- | ------------------------- |
| Soprano   | Flauta                    |
| Contralto | Oboé ou Clarinete         |
| Tenor     | Corne inglês ou Clarinete |
| Baixo     | Fagote                    |

**Fica bom para:** hinos calmos, delicados e introspectivos.
**Som esperado:** leve, doce e muito limpo.

**Cuidado:** o oboé pode chamar muita atenção. Use no contralto ou soprano com volume moderado.

---

## 6. Orquestra “hinário cantado” instrumental

| Voz       | Instrumento            |
| --------- | ---------------------- |
| Soprano   | Flauta + Violino suave |
| Contralto | Clarinete + Viola      |
| Tenor     | Trompa + Violoncelo    |
| Baixo     | Fagote + Cello grave   |

**Fica bom para:** gravações completas em MP3, com som mais profissional.
**Som esperado:** orquestra leve, sem exagero, com cada voz bem definida.

Essa é uma das melhores configurações para IA, porque mistura:

**Madeiras para clareza**
**Cordas para emoção**
**Trompa para corpo**
**Cello/fagote para base**

---

## 7. Piano + orquestra leve

| Função        | Instrumento       |
| ------------- | ----------------- |
| Harmonia base | Piano acústico    |
| Soprano       | Flauta ou Violino |
| Contralto     | Clarinete         |
| Tenor         | Viola             |
| Baixo         | Cello             |

**Fica bom para:** versões suaves, estudo, fundo musical e hinos mais lentos.
**Som esperado:** elegante, simples e bonito.

**Cuidado:** o piano não deve tocar forte demais. Ele deve apenas sustentar a harmonia.

---

## 8. Órgão + metais leves

| Voz       | Instrumento            |
| --------- | ---------------------- |
| Soprano   | Trompete suave         |
| Contralto | Trompa                 |
| Tenor     | Eufônio                |
| Baixo     | Tuba                   |
| Fundo     | Church Organ bem baixo |

**Fica bom para:** hinos bem solenes.
**Som esperado:** forte, cheio e com clima de culto.

Essa combinação fica bonita quando o órgão fica atrás, preenchendo, e os metais fazem as vozes principais.

---

## 9. Orquestra grande, mas sem exagero

| Voz       | Instrumentos                     |
| --------- | -------------------------------- |
| Soprano   | Flauta + Violino                 |
| Contralto | Clarinete + Viola                |
| Tenor     | Trompa + Cello                   |
| Baixo     | Fagote + Tuba leve + Cello grave |

**Fica bom para:** hinos de abertura, hinos fortes e gravações mais impactantes.
**Som esperado:** mais cheio, mas ainda reverente.

**Cuidado:** não usar percussão, prato, tímpano, bateria ou efeitos de cinema. Para hino da CCB, isso pode tirar o caráter sacro.

---

## 10. Versão bem tradicional com cara de CCB

| Voz       | Instrumento         |
| --------- | ------------------- |
| Soprano   | Clarinete ou Flauta |
| Contralto | Trompa              |
| Tenor     | Trombone/Eufônio    |
| Baixo     | Tuba                |

**Fica bom para:** hinos médios e fortes.
**Som esperado:** lembra banda/orquestra de igreja, com vozes bem separadas.

# Minha combinação favorita para IA

Para ficar bonito usando SF2, eu faria assim:

| Voz       | Instrumento principal | Instrumento de apoio     |
| --------- | --------------------- | ------------------------ |
| Soprano   | Flauta                | Violino suave            |
| Contralto | Clarinete             | Viola                    |
| Tenor     | Trompa                | Violoncelo               |
| Baixo     | Fagote                | Cello grave ou Tuba leve |

Essa combinação fica muito boa porque:

* a **flauta** destaca a melodia;
* o **clarinete** deixa o contralto macio;
* a **trompa** dá corpo ao tenor;
* o **fagote/cello** deixa o baixo firme;
* as **cordas** dão emoção sem pesar.

---

# Como dividir entre os SF2

Eu instruiria a IA assim:

| SF2                          | Melhor uso                                               |
| ---------------------------- | -------------------------------------------------------- |
| **SGM-V2.01.sf2**            | Base geral, cordas, metais equilibrados, madeiras limpas |
| **GeneralUser-GS.sf2**       | Som mais leve, estável e previsível                      |
| **CrisisGeneralMidi301.sf2** | Sons mais encorpados, bons para camadas e destaque       |

Uma boa regra seria:

**SGM-V2.01** para a maioria dos instrumentos.
**CrisisGeneralMidi301** para cordas mais bonitas ou instrumentos de destaque.
**GeneralUser-GS** para quando o som precisar ficar mais limpo e menos pesado.

# Prompt para sua IA

Você pode copiar e colar assim:

> Configure esta MIDI SATB de hino da Congregação Cristã no Brasil como uma orquestra sacra reverente, usando os SoundFonts `CrisisGeneralMidi301.sf2`, `GeneralUser-GS.sf2` e `SGM-V2.01.sf2`.
>
> Não quero som moderno, pop, cinema, bateria, percussão ou efeitos exagerados. O resultado deve parecer uma orquestra de igreja acompanhando um hino congregacional: limpo, equilibrado, solene e bonito.
>
> Teste os timbres dos três SF2 e escolha os mais naturais para cada função. Use preferencialmente `SGM-V2.01.sf2` como base geral, `CrisisGeneralMidi301.sf2` para cordas mais encorpadas e instrumentos de destaque, e `GeneralUser-GS.sf2` quando precisar de timbres mais limpos e leves.
>
> Distribua as vozes assim:
>
> Soprano: Flauta, Violino ou Trompete suave, sempre com a melodia clara, mas sem ficar agressiva.
> Contralto: Clarinete, Oboé suave, Viola ou Trompa, preenchendo a harmonia sem competir com o soprano.
> Tenor: Trompa, Viola, Violoncelo ou Eufônio, dando corpo ao meio da harmonia.
> Baixo: Violoncelo grave, Fagote, Tuba leve ou Pipe Organ grave, sustentando a base com firmeza, mas sem embolar.
>
> Para hinos suaves, priorize flauta, clarinete, viola, violoncelo e fagote.
> Para hinos solenes, use trompete suave, trompa, eufônio e tuba leve.
> Para hinos emocionais, use cordas completas com violino, viola, cello e contrabaixo leve.
> Para hinos com clima de culto, adicione Church Organ bem baixo no fundo, apenas para preencher, sem dominar a orquestra.
>
> Ajuste os volumes para manter o soprano levemente destacado, contralto e tenor equilibrados, e baixo firme. Use reverb de igreja moderado, pouco ou nenhum chorus, sem pitch bend, sem vibrato exagerado e sem humanização artificial excessiva.
>
> O objetivo final é gerar um MP3 com som de orquestra sacra tipo CCB, com vozes SATB bem separadas, harmonia clara, baixo definido e clima reverente.
